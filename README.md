# Three-Stage Mechanical Brake System for ROV Winch
**Mechanical Engineering Internship — Rovtech Solutions Ltd, Aberdeen, UK | June–July 2025**

## Overview

Designed and developed a manually operated three-stage band brake system for a Remotely Operated Vehicle (ROV) winch used in offshore wind operations. The brake controls pay-out and retrieval of a neutrally buoyant tether, providing three discrete states: free movement, controlled drag, and full mechanical lock.

This was a safety-critical engineering deliverable completed end-to-end during a live offshore wind project environment.

## Key Design Features

- **Two-shoe configuration** — each shoe spans 120° around a 100 mm diameter brass drum, lined with Kevlar for corrosion-resistant friction performance
- **Gear-and-pawl locking mechanism** — four asymmetric shark-tooth profiles replacing an initial notched plate design; provides secure indexing with no backlash under load
- **Multi-linkage actuation** — hand lever → lever rod → bell crank → triangular plate → dual pivot arms; fits flush within the winch frame envelope
- **Wear indicator** — concealed central countersunk screw fastens each Kevlar pad; screw head exposure signals pad replacement threshold
- **Marine-grade materials** — stainless steel shoes, brass drum, Kevlar linings selected for corrosion resistance under repeated marine splash exposure

## Force Analysis

| Parameter | Value |
|---|---|
| Input force (manual) | 200 N |
| Lever length | 284.85 mm |
| Input torque | 56.97 Nm |
| Linkage efficiency | 85% |
| Total clamping force | 1,177.77 N |
| Combined contact area | 0.021 m² |
| Mean contact pressure | 0.056 MPa |
| Kevlar operating limit | 0.5–1.0 MPa |

Analytical force modelling confirmed the system delivers reliable frictional clamping for tether drag control and emergency hold, operating well within Kevlar's contact pressure limits.

## Design Iterations

The design evolved through several key refinements:

1. **Lever locking** — slotted notched plate → curtain tensioner → gear-and-pawl, improving engagement reliability under fluctuating load
2. **Pawl geometry** — flat-tipped long pawl → shortened curved tip, reducing backlash and improving tooth engagement
3. **Brake lining attachment** — marine epoxy bond → concealed countersunk screw, eliminating long-term bond degradation risk and adding wear indication
4. **Linkage configuration** — single-rod → two-rod bell crank system, reducing overall assembly height to fit within winch frame constraints

## Constraints & Limitations

- Budget: £250–£350
- Frame envelope: 550 mm internal width, side-mounting footprint
- Clamping force sufficient for tether drag control; does not meet the 900–1,400 Nm target for full ROV static holding under worst-case conditions
- Physical testing was limited due to fabrication time constraints; validation conducted via analytical force modelling
